# Roadmap

## Alpha

- Skill pack for Capture Mode and Normal Mode
- Three-basket routing model
- Markdown inbox templates
- Basic install script
- Validation and secret-scan scripts
- Public-ready README and setup guide

## Beta

- Native Hermes plugin for `/capture` and `/normal`
- Optional `/today`, `/process`, and `/debrief` flows
- Cleaner calendar provider adapters
- Tests for skill metadata and config templates
- Demo screenshots or short video

## Public v1

- Clean install docs for macOS and Linux
- Optional plain-Markdown starter vault
- Package registry publishing if Hermes supports skill packs directly
- More example workflows from community feedback
