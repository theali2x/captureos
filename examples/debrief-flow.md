# Debrief Flow

Use this when the user says `Daily debrief` or `/debrief` if implemented.

Prompts:

1. What happened today?
2. Who did you speak to?
3. Any promises or follow-ups?
4. Any meetings/events/reminders to calendar?
5. Any ideas worth keeping?
6. Anything for work, clients, projects, or reusable IP?
7. Anything personally important?

Route outputs into:

- Task / Reminder
- Event / Meeting
- Idea / Note
