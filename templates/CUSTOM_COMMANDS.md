# Custom Operator Commands

Use this file as the source of truth for user-facing custom commands.

## CaptureOS

- `/capture` | normal-use | Enter Capture Mode.
- `/normal` | normal-use | Exit Capture Mode.

## Suggested future commands

- `/inbox` | planned | Show the three capture inboxes.
- `/today` | planned | Show today's calendar and open captures.
- `/process` | planned | Process messy notes into the three baskets.
- `/debrief` | planned | Start daily debrief flow.
